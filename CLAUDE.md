# 일본어 문법 교재 검수 프로젝트

## 임무
N5부터 N1까지 각 폴더의 JLPT N5~N1 교재 마크다운 파일들을 검수하고,
reports/ 폴더에 파일별 보고서를 작성한다.

## 작업 규칙 (반드시 준수)
1. 한 세션에서 반드시 **하나의 파일만** 처리한다.
2. 작업 시작 전에 PROGRESS.md를 읽어 다음 처리 대상 파일을 확인한다.
3. 보고서는 reports/{filename}_report.md에 저장한다.
4. 작업 완료 후 PROGRESS.md를 업데이트하고 종료한다.
5. 수정 작업은 검수 보고서가 완성된 파일에 한해서만 수행한다.

## 세션 처리 한도 (반드시 준수)
- 한 세션에서 처리하는 문법 항목은 최대 10개
- 챕터 내 항목 수가 10개를 초과하면 전반/후반으로 나눠 처리
- PROGRESS.md에 "Ch.X 전반 완료 (항목 1~8)" 형식으로 기록

## 파일 처리 순서
n5_part1 → n5_part2 → n4_part1 → n4_part2 → n3_part1 → n3_part2
→ n2_part1 → n2_part2 → n1_part1 → n1_part2 → n1_part3
