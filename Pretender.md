## Unit 1

### 1. 타임스탬프
- 00:31.95
- 00:35.17

### 2. 원문
君とのラブストーリー  
それは予想通り

### 3. 자연스러운 한국어 해석
너와의 러브스토리, 그건 예상한 대로였어.

### 4. 단어 정리
- **君** — 표제형: 君 / 읽기: きみ / 품사: 대명사 / 기본 뜻: 너 / 이 문장에서의 뜻: 상대방인 “너”
- **ラブストーリー** — 표제형: ラブストーリー / 읽기: ラブストーリー / 품사: 명사 / 기본 뜻: 러브스토리, 사랑 이야기 / 이 문장에서의 뜻: 화자와 상대의 관계를 가리키는 “사랑 이야기”
- **それ** — 표제형: それ / 읽기: それ / 품사: 대명사 / 기본 뜻: 그것 / 이 문장에서의 뜻: 앞의 “君とのラブストーリー”를 받는 말
- **予想** — 표제형: 予想 / 읽기: よそう / 품사: 명사 / 기본 뜻: 예상 / 이 문장에서의 뜻: 미리 짐작한 내용
- **通り** — 표제형: 通り / 읽기: どおり / 품사: 명사 / 기본 뜻: ~대로, ~와 같은 방식 / 이 문장에서의 뜻: 예상한 그대로

### 5. 문법/표현 포인트
- **〜との**
  - 형태: `君 + と + の`
  - 뜻: ~와의
  - 이 문장에서의 기능: `君とのラブストーリー` = “너와의 러브스토리”
  - 헷갈리기 쉬운 점: 조사 `と`를 단독으로 설명하는 것이 아니라, `との`가 하나의 연결 표현처럼 쓰여 “관계·소속”을 만든다.

- **名詞 + 通り**
  - 형태: `予想通り`
  - 뜻: 예상대로, ~한 그대로
  - 이 문장에서의 기능: 상황이 예상과 어긋나지 않았음을 나타낸다.
  - 헷갈리기 쉬운 점: `通り`는 “길”이라는 뜻도 있지만, 여기서는 “~대로”라는 뜻이다.

## Unit 2

### 1. 타임스탬프
- 00:38.03
- 00:42.42
- 00:44.84

### 2. 원문
いざ始まればひとり芝居だ  
ずっとそばにいたって  
結局ただの観客だ

### 3. 자연스러운 한국어 해석
막상 시작되고 나면 혼자 하는 연극이야.  
계속 곁에 있어 봤자 결국 그저 관객일 뿐이야.

### 4. 단어 정리
- **いざ** — 표제형: いざ / 읽기: いざ / 품사: 부사 / 기본 뜻: 자, 막상, 정작 / 이 문장에서의 뜻: “막상 실제로”
- **始まる** — 표제형: 始まる / 읽기: はじまる / 품사: 동사 / 기본 뜻: 시작되다 / 이 문장에서의 뜻: 관계나 이야기가 실제로 시작되다
- **ひとり芝居** — 표제형: ひとり芝居 / 읽기: ひとりしばい / 품사: 명사 / 기본 뜻: 1인극, 혼자 하는 연기 / 이 문장에서의 뜻: 상대와의 관계가 사실상 혼자만의 연극처럼 느껴지는 상태
- **ずっと** — 표제형: ずっと / 읽기: ずっと / 품사: 부사 / 기본 뜻: 계속, 줄곧 / 이 문장에서의 뜻: 오랫동안 계속
- **そば** — 표제형: そば / 읽기: そば / 품사: 명사 / 기본 뜻: 곁, 가까운 곳 / 이 문장에서의 뜻: 상대의 가까이
- **いる** — 표제형: いる / 읽기: いる / 품사: 동사 / 기본 뜻: 있다 / 이 문장에서의 뜻: 곁에 계속 머무르다
- **結局** — 표제형: 結局 / 읽기: けっきょく / 품사: 부사 / 기본 뜻: 결국 / 이 문장에서의 뜻: 결과적으로
- **ただ** — 표제형: ただ / 읽기: ただ / 품사: 표현 / 기본 뜻: 단지, 그저 / 이 문장에서의 뜻: 특별한 역할 없이 “그저”
- **観客** — 표제형: 観客 / 읽기: かんきゃく / 품사: 명사 / 기본 뜻: 관객 / 이 문장에서의 뜻: 관계의 중심이 아니라 지켜보기만 하는 위치

### 5. 문법/표현 포인트
- **動詞仮定形 + ば**
  - 형태: `始まれば`
  - 뜻: ~하면, ~하고 나면
  - 이 문장에서의 기능: “막상 시작되면”이라는 조건을 제시한다.
  - 헷갈리기 쉬운 점: `〜たら`보다 문어적·조건 제시 느낌이 조금 더 강하다.

- **〜たって**
  - 형태: `いた + って` → `いたって`
  - 뜻: ~해 봤자, ~해도
  - 이 문장에서의 기능: `ずっとそばにいたって` = “계속 곁에 있어 봤자”
  - 헷갈리기 쉬운 점: `〜ても`의 구어체·축약 느낌이다. 조사 `って`가 인용만 뜻하는 것은 아니다.

- **ただの + 名詞**
  - 형태: `ただの観客`
  - 뜻: 그저 ~일 뿐인
  - 이 문장에서의 기능: 화자의 위치를 “특별하지 않은 단순한 관객”으로 낮춘다.
  - 헷갈리기 쉬운 점: `ただ` 단독의 “단지”와 비슷하지만, `ただの`는 뒤 명사를 꾸민다.

### 6. 필요할 때만: 문맥/구조 보충
- **文脈上解釈**: `ひとり芝居`는 실제 연극이 아니라, 관계가 서로 주고받는 것이 아니라 혼자만 의미를 부여하는 상태를 비유적으로 압축한 표현이다.

## Unit 3

### 1. 타임스탬프
- 00:47.73
- 00:50.78
- 00:53.63

### 2. 원문
感情のないアイムソーリー  
それはいつも通り  
慣れてしまえば悪くはないけど

### 3. 자연스러운 한국어 해석
감정 없는 “I'm sorry”,  
그것도 늘 그렇듯 똑같고,  
익숙해지고 나면 나쁘지는 않지만.

### 4. 단어 정리
- **感情** — 표제형: 感情 / 읽기: かんじょう / 품사: 명사 / 기본 뜻: 감정 / 이 문장에서의 뜻: 진심 어린 감정
- **ない** — 표제형: ない / 읽기: ない / 품사: い형용사 / 기본 뜻: 없다 / 이 문장에서의 뜻: 감정이 결여되어 있다
- **アイムソーリー** — 표제형: アイムソーリー / 읽기: アイムソーリー / 품사: 표현 / 기본 뜻: I’m sorry / 이 문장에서의 뜻: 진심이 느껴지지 않는 사과 말
- **それ** — 표제형: それ / 읽기: それ / 품사: 대명사 / 기본 뜻: 그것 / 이 문장에서의 뜻: 앞의 “감정 없는 사과”
- **いつも** — 표제형: いつも / 읽기: いつも / 품사: 부사 / 기본 뜻: 언제나, 늘 / 이 문장에서의 뜻: 항상, 늘
- **通り** — 표제형: 通り / 읽기: どおり / 품사: 명사 / 기본 뜻: ~대로 / 이 문장에서의 뜻: 평소와 다름없는 상태
- **慣れる** — 표제형: 慣れる / 읽기: なれる / 품사: 동사 / 기본 뜻: 익숙해지다 / 이 문장에서의 뜻: 그런 상황에 무뎌지다
- **しまう** — 표제형: しまう / 읽기: しまう / 품사: 보조 동사 / 기본 뜻: 끝내 버리다, 완전히 그렇게 되다 / 이 문장에서의 뜻: 결국 익숙해져 버리다
- **悪い** — 표제형: 悪い / 읽기: わるい / 품사: い형용사 / 기본 뜻: 나쁘다 / 이 문장에서의 뜻: 썩 좋지는 않지만 아주 나쁘지도 않다

### 5. 문법/표현 포인트
- **名詞 + の + ない**
  - 형태: `感情のない`
  - 뜻: 감정이 없는
  - 이 문장에서의 기능: `アイムソーリー`를 수식하며 “감정 없는 사과”라는 뜻을 만든다.
  - 헷갈리기 쉬운 점: `ない`는 여기서 단순 부정이라기보다 “결여”의 의미가 강하다.

- **名詞 + 通り**
  - 형태: `いつも通り`
  - 뜻: 평소대로
  - 이 문장에서의 기능: 사과가 특별하지 않고 늘 같은 방식임을 나타낸다.
  - 헷갈리기 쉬운 점: Unit 1의 `予想通り`와 같은 구조지만, 기준이 “예상”이 아니라 “평소”다.

- **〜てしまう**
  - 형태: `慣れてしまえば`
  - 뜻: ~해 버리다, 결국 ~하게 되다
  - 이 문장에서의 기능: 원치 않더라도 어느새 익숙해져 버린 상태를 나타낸다.
  - 헷갈리기 쉬운 점: 완료뿐 아니라 “유감·체념”의 뉘앙스가 자주 붙는다.

- **〜ば**
  - 형태: `慣れてしまえば`
  - 뜻: ~하면
  - 이 문장에서의 기능: “익숙해지면”이라는 조건을 만든다.
  - 헷갈리기 쉬운 점: `〜てしまう`와 합쳐져 하나의 덩어리처럼 읽어야 한다.

- **〜はない**
  - 형태: `悪くはない`
  - 뜻: 나쁘지는 않다
  - 이 문장에서의 기능: 전면 긍정은 아니고, 부정을 약하게 눌러 말한다.
  - 헷갈리기 쉬운 점: 단순 `悪くない`보다 대비·완곡의 느낌이 더 있다.

- **〜けど**
  - 형태: `悪くはないけど`
  - 뜻: ~이지만, ~인데
  - 이 문장에서의 기능: 뒤에 이어질 마음의 유보를 남긴다.
  - 헷갈리기 쉬운 점: 가사에서는 뒤 절을 생략하고 여운만 남기는 경우가 많다.

## Unit 4

### 1. 타임스탬프
- 00:58.02
- 01:01.60

### 2. 원문
君とのロマンスは人生柄  
続きはしないことを知った

### 3. 자연스러운 한국어 해석
너와의 로맨스는, 삶이라는 것의 성질상 오래 이어지지 않는다는 걸 알았어.

### 4. 단어 정리
- **君** — 표제형: 君 / 읽기: きみ / 품사: 대명사 / 기본 뜻: 너 / 이 문장에서의 뜻: 상대방
- **ロマンス** — 표제형: ロマンス / 읽기: ロマンス / 품사: 명사 / 기본 뜻: 로맨스 / 이 문장에서의 뜻: 연애, 사랑의 관계
- **人生柄** — 표제형: 人生柄 / 읽기: じんせいがら / 품사: 표현 / 기본 뜻: 삶의 성질, 인생이라는 것의 특성 / 이 문장에서의 뜻: “삶이란 본래 그런 것”이라는 식의 가사체 압축 표현
- **続く** — 표제형: 続く / 읽기: つづく / 품사: 동사 / 기본 뜻: 계속되다 / 이 문장에서의 뜻: 관계가 이어지다
- **こと** — 표제형: こと / 읽기: こと / 품사: 명사 / 기본 뜻: 일, 것 / 이 문장에서의 뜻: “계속되지 않는다”는 사실 자체
- **知る** — 표제형: 知る / 읽기: しる / 품사: 동사 / 기본 뜻: 알다 / 이 문장에서의 뜻: 깨닫게 되다

### 5. 문법/표현 포인트
- **連用形 + はしない**
  - 형태: `続きはしない`
  - 뜻: 결코 계속되지는 않는다, 이어지지는 않는다
  - 이 문장에서의 기능: `続く`를 단순 부정하는 것이 아니라 “이어지는 일은 없다”는 식으로 강하게 부정한다.
  - 헷갈리기 쉬운 점: `続く`의 연용형 `続き`가 명사처럼 보이지만, 여기서는 `〜はしない`와 결합한 동사적 표현이다.

- **普通形 + こと**
  - 형태: `続きはしないこと`
  - 뜻: 계속되지 않는다는 것
  - 이 문장에서의 기능: 문장 전체를 명사처럼 만들어 `知った`의 목적어로 쓴다.
  - 헷갈리기 쉬운 점: `こと`는 “일”이 아니라 앞 문장을 통째로 내용화한다.

### 6. 필요할 때만: 문맥/구조 보충
- **文脈上解釈**: `人生柄`는 일상적으로 아주 흔한 표현은 아니고, 가사체로 압축된 말이다. `人生の性質上`, `人生というものの成り行き上` 정도로 이해하면 된다.

- **〜そう**
  - 형태: `偉そう`
  - 뜻: ~해 보이는, ~한 듯한
  - 이 문장에서의 기능: 실제로 “위대하다”가 아니라 “잘난 척해 보이는” 태도를 나타낸다.
  - 헷갈리기 쉬운 점: `おいしそう` 같은 단순 추측과 달리, `偉そう`는 굳어진 표현으로 “거만한 듯한” 뜻이 강하다.

- **何ひとつとして〜ない**
  - 형태: `何ひとつとして ... こなくて`
  - 뜻: 무엇 하나도 ~않다
  - 이 문장에서의 기능: 와닿는 것이 전혀 없음을 강하게 말한다.
  - 헷갈리기 쉬운 점: `何も〜ない`보다 강조가 더 세다.

- **〜なくて**
  - 형태: `こなくて`
  - 뜻: 오지 않아서, 와닿지 않아서
  - 이 문장에서의 기능: 뒤의 비유 표현으로 이어지는 이유를 만든다.
  - 헷갈리기 쉬운 점: 단순 부정 연결형으로, 여기서는 “그래서 ~같다”로 이어진다.

- **動詞た形 + 名詞**
  - 형태: `見下ろした夜景`
  - 뜻: 내려다본 야경
  - 이 문장에서의 기능: 앞 줄 `見下ろした`가 뒤 줄의 `夜景`를 꾸민다.
  - 헷갈리기 쉬운 점: 줄이 나뉘어 있어도 하나의 수식 구조다.

- **〜みたいだ**
  - 형태: `夜景みたいだ`
  - 뜻: ~같다
  - 이 문장에서의 기능: 전체 인상을 비유한다.
  - 헷갈리기 쉬운 점: `ようだ`보다 구어적이고 부드럽다.

### 6. 필요할 때만: 문맥/구조 보충
- **省略**: `飛行機の窓から見下ろした` 뒤에 바로 `知らない街の夜景`가 이어져, “비행기 창문에서 내려다본 낯선 도시의 야경”이라는 하나의 명사구를 만든다.
- **文脈上解釈**: `ピンとこない`는 직역하면 “팽 하고 오지 않는다”가 아니라 “감이 오지 않다, 와닿지 않다”로 이해해야 자연스럽다.

  - 이 문장에서의 기능: 반복할수록 같은 결과가 생김을 나타낸다.
  - 헷갈리기 쉬운 점: 단순 시간 표현이 아니라 반복적 상관관계다.

- **〜だす**
  - 형태: `疼きだす`
  - 뜻: ~하기 시작하다
  - 이 문장에서의 기능: 미래가 “욱신거리기 시작하는” 상태를 나타낸다.
  - 헷갈리기 쉬운 점: `始める`보다 붙어서 쓰이며, 갑작스러운 시작 느낌이 있다.

- **〜には**
  - 형태: `未来には`
  - 뜻: 미래에는
  - 이 문장에서의 기능: 미래를 화제로 삼으면서 대조를 만든다.
  - 헷갈리기 쉬운 점: 조사 `に`와 `は`가 결합해, 단순 위치보다 대비가 강해진다.

- **そりゃ**
  - 형태: `そりゃ`
  - 뜻: 그야, 당연히
  - 이 문장에서의 기능: 뒤의 `苦しい`를 당연한 감정으로 받는다.
  - 헷갈리기 쉬운 점: `それは`의 구어 축약형이다.

### 6. 필요할 때만: 문맥/구조 보충
- **文脈上解釈**: `エンドライン`은 실제 선이 아니라 관계의 끝을 비유적으로 말한 것이다.

### 5. 문법/표현 포인트
- **それもこれも**
  - 형태: `それもこれも`
  - 뜻: 이것도 저것도, 이 모든 것
  - 이 문장에서의 기능: 앞서 나온 여러 상황을 한꺼번에 묶는다.
  - 헷갈리기 쉬운 점: 각각을 따로 가리키기보다 “전부 다”의 느낌이 강하다.

- **名詞 + なら**
  - 형태: `定めなら`
  - 뜻: ~이라면
  - 이 문장에서의 기능: “그게 사랑의 운명이라면”이라는 가정을 만든다.
  - 헷갈리기 쉬운 점: `なら`는 앞 상황을 받아 판단·평가를 잇는 데 자주 쓰인다.

- **〜よな**
  - 형태: `悪くないよな`
  - 뜻: 나쁘지 않지, 그렇지?
  - 이 문장에서의 기능: 혼잣말처럼 자기 판단을 확인한다.
  - 헷갈리기 쉬운 점: `よ`의 단정과 `な`의 확인/여운이 함께 있다.

- **名詞も名詞もない**
  - 형태: `永遠も約束もない`
  - 뜻: 영원도 약속도 없다
  - 이 문장에서의 기능: 중요한 두 요소가 모두 없음을 병렬적으로 강조한다.
  - 헷갈리기 쉬운 점: `も`가 긍정의 “~도”가 아니라, 부정문에서 “~도 ~도 없다”의 전면 부정이 된다.

- **〜けれど**
  - 형태: `ないけれど`
  - 뜻: 없지만
  - 이 문장에서의 기능: 결핍을 인정하면서도 뒤에서 긍정 평가 `綺麗だ`로 이어진다.
  - 헷갈리기 쉬운 점: `けど`보다 약간 문어적이지만 기능은 비슷하다.

### 6. 필요할 때만: 문맥/구조 보충
- **文脈上解釈**: `定め`는 단순 “규정”이 아니라, 여기서는 “피할 수 없는 운명” 쪽 의미다.
