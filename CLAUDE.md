# 일본어 문법 교재 검수 프로젝트

## 파일 구성
N5부터 N1꺼지의 폴더 5개에 JLPT N5~N1 교재 마크다운 파일 총 11개가 있다.
처리 순서: N5_상편 → N5_하편 → N4_상편 → N4_하편 → N3_상편 → N3_하편
         → N2_상편 → N2_하편 → N1_상편 → N1_중편 → N1_하편

## 절대 규칙 (위반 금지)
1. 한 세션에서 처리하는 문법 항목은 최대 10개
2. 챕터 항목이 10개 초과 시 전반/후반으로 분할하여 각각 별도 세션에서 처리
3. 보고서는 반드시 표 형식으로만 작성 (서술형 금지)
4. 새 내용을 보고서 파일 끝에 append만 한다 (전체 재작성 금지)
5. 한 세션 작업 완료 후 반드시 PROGRESS.md를 업데이트하고 즉시 종료

## 세션 시작 절차
1. PROGRESS.md를 읽어 다음 처리 대상과 범위를 확인
2. evaluation_criteria.md를 읽어 검수 기준 숙지
3. 지정된 범위만 처리 (다음 챕터·파일로 절대 넘어가지 않음)

## 보고서 형식 (이 형식 외 사용 금지)
### [파일명] Ch.X [챕터 제목] (항목 N~M)
| 항목 | 줄번호 | 유형 | 문제 내용 | 수정안 |
|------|--------|------|-----------|--------|
이슈 없으면: "이슈 없음" 한 줄만 기재

- [x] N5_상편 Ch.1 후반 (항목 7~9) — 검수완료
- [ ] N5_상편 Ch.2 — 미완료
